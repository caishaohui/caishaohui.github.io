gulp使用总结

  1. 首先需要全局安装gulp,如果之前已经全局安装过,可以省略这步.
    npm install gulp -g win r键

  2. 本地安装,在项目目录中.
    先cd 到要安装的目录中用shift加鼠标右键页可以
    npm install gulp –save

  3. 在项目目录中新建1个gulpfile.js文件.
    在这个文件中写上构建代码.

  4. 下载 压缩css包 npm install gulp-cssmin –save
    下载 压缩js包 npm install gulp-uglify –save
    下载 代码合并包 npm install gulp-concat –save
    下载 压缩html包 npm install gulp-htmlmin –save

  5. 引包
    var gulp = require(“gulp”);

    var cssmin = require(“gulp-cssmin”);
    //cssmin是1个函数. css包
    //绝大多数的关卡都是1个函数.

    var uglify = require(“gulp-uglify”);
    //是1个函数. js包

    var concat = require(“gulp-concat”);
    //是1个函数 合并包

    var htmlmin = require(“gulp-htmlmin”);
    //函数 html包

  6. 压缩css 判断是否要排除第三方文件
    gulp.task(“yscss”, function () {
    gulp.src(“./src/*/.css”)

    });

  7. 执行cmd命令 gulp yscss alt+z对压缩文件自动换行

  8. 合并和 压缩js 排除第三方文件,合并自己写的js代码,体积小的合并
    要注意每个js文件开头和结尾要用 ; 隔开
    gulp.task(“ysjs”, function () {
    gulp.src([“./src/*/.js”, “!./src/assets/js/prefixfree.min.js”])

    });

  9. 执行cmd命令 gulp ysjs alt+z对压缩文件自动换行

  10. 改html引入的js文件

  11. 压缩html代码 判断是否要排除第三方文件
    gulp.task(“yshtml”, function () {
    gulp.src(“./src/*/.html”)

    });

  12. 执行cmd命令 gulp yshtml 打开文件alt+z对压缩文件自动换行

  13. 有一些排除的文件没移动到压缩文件,需要手动的把这些文件移到对应的目录,列如
    img文件夹,排除的文件,或者第三方文件等
